# gethash/homebrew-tap

Homebrew formulae for [gethash](https://github.com/gethash) tools.

## Install the tap

```sh
brew tap gethash/tap
```

## Formulas

| Formula | Description | Source |
|---------|-------------|--------|
| [bamboozle](Formula/bamboozle.rb) | Fullscreen PDF presenter with auto-advance, spiritual successor to Impressive | [gethash/bamboozle](https://github.com/gethash/bamboozle) |

## Usage

```sh
brew install gethash/tap/<formula>
```

For example:

```sh
brew install gethash/tap/bamboozle
```

## Platform notes

- **macOS** — requires Xcode Command Line Tools (`xcode-select --install`); no additional dependencies needed.
- **Linux** — Mesa and X11 extension libraries are pulled in automatically by Homebrew.
